Surat Masuk

Cara input dan proses surat masuk.

  1. Buka menu Input Surat Masuk.
  2. Isi Tanggal Surat, Pengirim, dan Perihal.
  3. Nomor Agenda akan terisi otomatis.
  4. Upload file (PDF/Gambar). Klik Simpan.
  5. Surat akan masuk ke Inbox Direktur untuk disposisi awal.
Surat Keluar

Booking nomor dan approval.

  1. Buka menu Input Surat Keluar.
  2. Klik tombol Booking jika belum ada nomor fisik, atau ketik manual.
  3. Pilih Pembuat Surat (Bagian/Bidang).
  4. Status awal adalah "Pending Approval".
  5. Admin akan mengupdate status setelah TTD pimpinan.
Fitur Delegasi

Mengakses Inbox Atasan/Bawahan.

  • Jika Anda diberi hak akses (misal: Sekretaris mengakses Inbox Direktur), menu akan muncul di sidebar kiri bawah.
  • Klik nama delegasi untuk masuk ke inbox mereka.
  • Setiap tindakan (disposisi/selesai) akan tercatat "a.n." (atas nama).
Pencarian & Arsip

Mencari surat lama.

  • Gunakan Pencarian Lanjut di sidebar.
  • Bisa filter berdasarkan Tanggal, Kata Kunci, atau Kategori.
  • Menu Arsipku berisi surat yang sudah Anda tandai "Selesai".
Logika Sistem

Sistem menggunakan hierarki bertingkat. Surat tidak bisa langsung lompat dari Direktur ke Staf, harus melalui Wadir dan Kabid/Kabag terlebih dahulu.

1. Registrasi (Admin)
Admin menginput surat di Loket. Status: Menunggu Disposisi.
2. Direktur
Menerima surat. Memilih disposisi ke Wadir (Umum/Pelayanan).
3. Wadir
Menerima dari Direktur. Meneruskan ke Kepala Bagian / Kepala Bidang terkait.
4. Kabag / Kabid
Menerima dari Wadir. Meneruskan ke Staf Pelaksana atau Kepala Ruangan.
5. Staf Pelaksana
Menerima instruksi akhir. Melakukan tindakan dan klik tombol Selesai (Centang Hijau). Surat masuk ke Arsipku.

Hanya user dengan hak akses khusus (Admin/Super Admin) yang bisa mengedit tujuan disposisi. Buka menu Arsip Masuk, cari suratnya, lalu klik tombol (Edit Disposisi) di kolom Aksi.

1. Pastikan ukuran file di bawah 5MB.
2. Pastikan format file adalah PDF, JPG, atau PNG.
3. Cek koneksi internet Anda.
Jika masih gagal, mungkin server penyimpanan sedang penuh/dialihkan. Hubungi IT untuk cek status "Gudang Penyimpanan" di menu Pengaturan.

Klik tombol (Mata) pada daftar surat (Inbox atau Arsip). Akan muncul Timeline History lengkap mulai dari surat diterima, didisposisi oleh siapa, jam berapa, hingga siapa yang menyelesaikan.

Menu Input dibatasi hak aksesnya. Biasanya hanya Admin Loket atau Sekretariat yang memilikinya. Jika Anda membutuhkan akses, hubungi Super Admin untuk mengedit "Permissions" akun Anda di menu Pengaturan User.
